# Private Pensieve Product Specification ## Product Promise Private Pensieve is an offline, private AI memory companion. Users speak or type thoughts, save them locally, and later recall what they shared. ## Key Constraints - No login, cloud, social, ads, or analytics. - Everything stays on-device. - Midnight Sanctuary aesthetic: Warm, minimal, dark. ## Screen List 1. Onboarding 1: Welcome & Promise 2. Onboarding 2: Privacy Mechanics (Local-only) 3. Onboarding 3: Passcode & Biometrics 4. Onboarding 4: AI Model Selection 5. Talk (Idle): Orb identity, "I'm here", Mic button 6. Talk (Listening): Restrained waveform, transcription preview 7. Transcript Review: Editable surface, "Save as memory" 8. Vault: Search, Filters, Memory Cards 9. Memory Detail: Full content, tags, actions 10. Recall (Search): Question input, suggested questions 11. Recall (Answer): Local-answer card, evidence cards 12. Privacy: Status rows (Cloud: Not used, etc.), Export/Delete actions 13. Success/Empty/Error States: "Your vault is waiting", No-evidence message.
