# UI/UX Specification — Baseline

## Product feeling
Private, calm, human, minimal, non-medical, non-addictive.

## Primary navigation
Four tabs only:
1. Talk
2. Vault
3. Recall
4. Privacy

## Screen: Talk
- Top local-only/offline badge
- Center: calm animated orb, no anthropomorphic face
- Primary action: large `Hold to Speak` or tap-to-toggle control
- Live transcript region
- States: idle, listening, transcribing, thinking, speaking, saved
- Toggle: save memory (default on); retain audio (default off)
- Always show: `Your memories stay on this device.`

## Screen: Transcript Review
- Editable transcript
- Buttons: Save memory, Edit, Discard
- Clarify what will be saved: compact memory card + transcript; audio only if enabled

## Screen: Vault
- Card list with title, date, one-line summary, calm tags
- Simple filters: Important, This Week, Goals, People, Ideas, Feelings
- No endless feed or engagement counters

## Screen: Recall
- Voice/text question input
- Suggested questions:
  - What did I say about my goals?
  - When was I feeling stressed?
  - What made me happy recently?
  - What did I promise myself?
- Answer shows 1–3 evidence cards beneath response

## Screen: Privacy
- Explicit status list:
  - Account: not required
  - Server: not used
  - Cloud: not used
  - AI: on-device
  - Storage: encrypted locally
  - Audio: retained only if enabled
- Actions: export, import, delete memories, delete audio, reset vault

## Accessibility
- Dynamic text sizing
- VoiceOver/TalkBack labels
- High contrast mode
- Reduced motion support
- Clear microphone permission explanation
