# Eight-Week Release Plan

## Week 1 — Contracts and native shells
Docs finalized, CI created, native app shells build, navigation and privacy baseline.

## Week 2 — Encrypted vault
Schema, secure key architecture, repositories, biometric/passcode gate, deletion semantics.

## Week 3 — Voice
Recording, transcript review, local STT abstraction, local TTS.

## Week 4 — Memory ingestion
Memory cards, scores, daily summary, duplicate grouping, fixture tests.

## Week 5 — Recall
Hybrid retrieval, ranking, context compression, evidence-bound answers.

## Week 6 — On-device AI adapters
iOS/Android adapters behind interfaces; unavailable-state UX; offline integration.

## Week 7 — Backup, privacy, performance
Encrypted export/import, privacy dashboard, offline test matrix, storage/latency profiling.

## Week 8 — Release candidate
Bug bash, accessibility, license/privacy audit, TestFlight and Play Internal Testing builds, release notes.

## Scope protection
If a feature threatens release, remove it rather than weakening privacy or offline guarantees.
